# Electron UI

Real-time telemetry monitor and daemon control.

## Setup

```bash
cd web
yarn install
```

## Development

```bash
yarn dev
```

This starts both the Vite dev server and Electron window.

## Build

```bash
yarn build
```

## Features

- Real-time telemetry visualization
- G-Force graph & orientation display
- Impact event detection
- Daemon status monitoring
- Configuration UI
